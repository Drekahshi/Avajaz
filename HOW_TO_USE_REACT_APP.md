# AVAJAZ - Gamified Impact Network (React dApp)
Welcome to the AVAJAZ React application! This modern, web3-enabled frontend allows you to interact with the Avalanche Fuji Testnet and participate in our gamified impact verification network.

## 🚀 How to Run the App

### Requirements
- Node.js installed (v18+)
- An Avalanche-compatible web3 wallet (like Core or MetaMask) installed in your browser.

### Starting the Development Server
1. Open a terminal and navigate to the `frontend` directory:
   ```bash
   cd c:\Users\Austin NAMUYE\OneDrive\Desktop\AVAJAZ\frontend
   ```
2. Install the necessary dependencies (only needed the first time):
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
4. Open your browser and navigate to the local URL provided by Vite (typically `http://localhost:5173`).

## 🎮 How to Use the App

### 1. The Landing Page
When you open the app, you will see the AVAJAZ landing page outlining the core mission: transforming stewardship work into verifiable digital assets. 
You will see 4 main categories displayed:
- **🌿 Seedling Nursery**
- **🌳 Agroforestry**
- **🍯 Beekeeping**
- **🦜 Ecotourism**

### 2. Connect Your Wallet
To access the Dashboard and participate in the network, you must connect your Web3 wallet.
1. Click the large **"Connect Wallet"** button on the landing page.
2. Your browser wallet extension (e.g., Core or MetaMask) will prompt you to connect.
3. Once connected, your wallet address will be read, and you will be automatically redirected to the **Dashboard**.

*(Note: The app is built on the Avalanche Fuji Testnet. Ensure your wallet is configured for the Fuji network to interact properly with future smart contracts.)*

### 3. The Dashboard
The dashboard is your secure portal to the AVAJAZ ecosystem.
- **Protected Routing:** The dashboard is protected. If you disconnect your wallet or try to access `/dashboard` directly without connecting, you will be safely redirected back to the landing page.
- **Your Hub:** From here, you can view the various tasks, participate in the gamified verification process, and manage your impact rewards!
